# Iniciar projeto

Para iniciar um projeto com node + typescript

## 1. init

```bash

npm init -y

# isso vai criar o package.json (onde encontra-se as libs e ferramentas instaladas no projeto.)
```

## 2. install

```bash

npm install -D # libs/ferramentas que deseja no projeto para dev

npm install # libs/ferramentas que deseja como dependencias no projeto

```
